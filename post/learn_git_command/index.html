1 min read

Since Git transfer data with SSH. We should configue the SSH in local machine

ssh-keygen -t rsa -C "youremail@example.com"

After this command, in .ssh folder, you will find two files: id_rsa and id_rsa.pub. Private Key and Public Key

Submmit public key info to GitHub

Combine local repo with remote repo in GitHub

git remote add origin https://github.com/AllyBei/LearnGitCommand.git

First time push master(local) to origin(remote) You have to add -u

git push -u origin master

In the future

git push origin master
