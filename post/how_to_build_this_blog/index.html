How to build this blog in github with HUGO

Dec 3, 2019
1 min read

Prepare

Intsall HUGO

The network is crazy in China… Failed many times to install via source code.

cd $(go env GOPATH)
cd src/github.com
git clone https://github.com/gohugoio/hugo.git
cd hugo
go install ./...

recommend to add &GOPATH/bin in .bashrc

hugo version

if this command is sucessful, HUGO is installed sucessfully

New a HUGO blog site in local environment

navigate to the folder you want to build HUGO site The site name I use “myblog”

hugo new site myblog

Then you should select a theme I choose m10c here. Make sure you are in the blog site folder

git clone https://github.com/vaga/hugo-theme-m10c.git themes/m10c

New a blog post with name whatever.

hugo new post/introduce.md
cd content/post/

Write your own blog md file

Make sure you have back to your blog site root path

hugo server -t m10c --buildDrafts

Build your own github public repo

New a public repo with name “{You own GitHub username}.github.io”

Package and push your local blog to GitHub

hugo --theme=m10c --baseUrl="https://allybei.github.io" --buildDrafts

a folder named “public” is created

cd public/
git init
git add *
git commit -m "My First Blog"
git remote add origin https://github.com/AllyBei/allybei.github.io.git
git push -u origin master
